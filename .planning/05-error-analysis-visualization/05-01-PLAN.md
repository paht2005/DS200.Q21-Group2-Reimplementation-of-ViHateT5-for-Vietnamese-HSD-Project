Add two new functions after `statistical_significance_report`:

1. `mcnemar_test(y_true, preds_a, preds_b)` — Computes McNemar's chi-squared statistic with continuity correction. Parameters: y_true (list/array), preds_a (list/array from model A), preds_b (list/array from model B). Returns tuple: (chi2_statistic: float, p_value: float, n01: int, n10: int). Uses contingency table: n01 = A correct & B wrong, n10 = A wrong & B correct. Formula: chi2 = (|n01 - n10| - 1)^2 / (n01 + n10). Edge case: if n01 + n10 == 0, return (0.0, 1.0, 0, 0). P-value from chi2_dist.sf(chi2, df=1). Also flag unreliable results when n01 + n10 < 25.

2. `mcnemar_report(test_pairs, y_true, predictions_dict, task_name, filename="mcnemar_results.csv")` — Runs mcnemar_test for each pair in test_pairs (list of tuples: (model_a_name, model_b_name)). predictions_dict maps model_name → list of predicted labels. Builds a DataFrame with columns: model_a, model_b, task, chi2, p_value, n01, n10, significant (bool, p < 0.05), reliable (bool, n01+n10 >= 25). Saves to ANALYSIS_DIR/filename. Prints results table. Returns the DataFrame.
